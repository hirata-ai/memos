# ユーザを追加する
## `useradd` コマンド
* `useradd (作成したいユーザ名)` で追加できる．（sudo必要）
* パスワード指定するときは`-p`オプションをつける

# パスワードを変更する
## `passwd` コマンド
* `passwd` で一般ユーザの場合自身のパスワードを変更できる．
* rootで一般ユーザのパスワードを変更する場合，`passwd (ユーザ名)`でパスワードを変更できる．
* あまりにも適当なパスワード(1234など)を指定すると怒られるっぽい

# sudo ユーザに追加する
* `visudo` でsudoers ファイルにユーザを追加する  
```
root      ALL=(ALL)       ALL
+ adduser  ALL=(ALL)       ALL
```
